CSS - Cascading Style Sheets

Used to enhance the look of a web page

Where to write css

  • Inline using style attribute
  • Internal--using style element
  • External--using a linked .css file (Recommended)

CSS syntax 

        
            selector{
                property: value
            }

Selector --helps target an element or a group of elements

{} Declaration block-- groups styles ( property value pairs) to be applied on the selected element/elements

Property--a presentation attribute that can be changed eg color

Value-- property specific varaible that changes the look of an element

CSS Selectors

  • Element name selector e.g p, h2, li
  • ID selector -- use id attribute (used to target 1 element)
  • Class selector ----targeted with a dot (targets a group of elements with same class)
